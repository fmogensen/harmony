# /harmony-apply

Reconcile your machine to your harmony manifest. Installs missing plugins, marketplaces, MCP servers, brew deps, and LaunchAgents — and propagates removals.

This is the same as running `harmony apply` in your terminal, but the output lands in the chat for you to see.

```bash
harmony apply
```
